Session State Machine

The target session workflow is an explicit graph. UI commands and guide proposals use the same graph-validated actions.

idleNo active session workflow.
target_selectionChoose guide-prioritized active target.
preparationUpdate and approve assessment.
stimulationRun set, log set, pause, or close.
interjectionGrounding and pause between sets.
closureClose down or return to stimulation.
reviewReview final data before ending.
post_sessionReturn idle or start another session.
